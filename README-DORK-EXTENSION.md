# Dork Link Collector (Browser Extension)

هذه الإضافة تجمع روابط من صفحات نتائج محركات البحث (Google, Bing, DuckDuckGo, Yahoo) باستخدام قائمة dorks تقوم بإدخالها محليًا في المتصفح. لا تحتاج أي خادم أو مفتاح API — كل العمل يتم محليًا عبر فتح صفحات النتائج واقتناص الروابط من DOM.

ملاحظة مهمة حول الاستخدام:
- استعمال هذه الأداة يجب أن يكون وفق قوانين محركات البحث وسياسات الاستخدام. لا تستخدمها لأغراض غير قانونية.
- فتح كثير من الصفحات بسرعة قد يؤدي إلى ظهور CAPTCHA أو حظر مؤقت من محرك البحث. استخدم خيار "تأخير بين الطلبات" لتخفيف المخاطر.

كيفية التثبيت (Chrome / Edge):
1. اذهب إلى chrome://extensions
2. فعل وضع المطور (Developer mode).
3. اختر "Load unpacked" وحدد المجلد الذي يحتوي على الملفات في هذا الفرع (`feat/dork-extension`).

كيفية الاستخدام:
1. افتح الأداة من شريط الإضافات.
2. ألصق Dorks (سطر لكل dork)، اضبط "نتائج لكل dork"، و"الحد الأقصى للروابط"، وحدد محرك البحث.
3. اضغط "ابدأ"، انتظر حتى تجمع الإضافة روابط، ثم استخدم "تنزيل (.txt)" لحفظها.

تحميل ZIP:
بعد دفع الملفات إلى الفرع `feat/dork-extension` يمكنك تنزيل كحزمة ZIP عبر الرابط التالي (GitHub يقدم تنزيل للفرع):
